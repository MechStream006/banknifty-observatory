# ops/runbooks/ — Operational Runbooks

Step-by-step procedures for every failure scenario identified in the infrastructure blueprint. Implemented in Milestone 16 (Operational Hardening).

Each runbook covers: detection, diagnosis, recovery steps, validation, and incident closure (service restored AND dataset annotated).
